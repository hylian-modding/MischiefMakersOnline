# MischiefMakersOnline

## MischiefMakersOnline
This is an online mod for ModLoader64.

## Features
Seeing other players.
Sharing Story Progression
Sharing Gold Gems
Sharing Best Times
Discord Integration

## Building
1. Install the ModLoader64 SDK.
2. Clone this repository.
3. modloader64 -nbd
